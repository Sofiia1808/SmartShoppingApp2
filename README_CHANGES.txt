Оновлення SmartShoppingApp

Що змінено:
1. Нижня вкладка "Рекомендації" перейменована у "Поради", щоб текст поміщався в один рядок.
2. Блок "Як це працює" прибрано.
3. Додано welcome / login / register екрани.
4. Додано локальні акаунти користувачів через Room + збереження сесії.
5. Товари тепер не заповнюються автоматично. Після входу список порожній.
6. Покупки реально додаються у базу даних і прив'язані до конкретного користувача.
7. Додано екран кабінету з профілем, виходом з акаунта та розумними порадами.
8. Додано більш м'який сучасний дизайн карток та welcome-екрана.

Примітка:
- Проєкт відкривай в Android Studio і виконай Gradle Sync.
- У початковому архіві був відсутній gradle-wrapper.jar, тому збірку найзручніше запускати саме через Android Studio.
