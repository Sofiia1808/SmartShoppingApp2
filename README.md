# SmartShoppingApp

Мобільний застосунок для обліку, аналізу та оптимізації покупок з персональними рекомендаціями.

## Що реалізовано
- Kotlin + Jetpack Compose
- Room (SQLite) через KSP
- MVVM
- Історія покупок
- Фільтрація за категорією
- Аналітика витрат
- Блок рекомендацій на основі середнього інтервалу між покупками
- Локальні нагадування через WorkManager

## Як відкрити в Android Studio
1. Розпакуйте архів.
2. Відкрийте папку `SmartShoppingApp` як проєкт в Android Studio.
3. Дочекайтесь синхронізації Gradle.
4. Якщо Android Studio запропонує оновити Gradle wrapper або створити локальні файли — погодьтесь.
5. Запустіть застосунок на емуляторі або телефоні.

## Примітка
У контейнері я не можу зібрати APK, бо тут немає Android SDK. Тому я підготував повний код проєкту та структуру файлів для відкриття в Android Studio.
